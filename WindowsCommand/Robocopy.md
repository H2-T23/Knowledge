# Robocopy

## [Syntax]

```
	robocopy $(source)$(destination)[<File> [...][<options>]	
```


## [Parameters]

|Parameter	|										|
|:----------|:--------------------------------------|
|source		|ソースディレクトリパス					|
|destination|インストール先ディレクトリパス			|
|File		|コピーするファイル(ワイルドカード)		|
|options	|Robocopyコマンドを使用するオプション	|


### Copy Options

|options	|										|
|:----------|:--------------------------------------|
|/s			|サブディレクトリをコピー				|
|/e			|サブディレクトリをコピー				|
|/lev:<N>	|コピー元ディレクトリツリーの上位Nレベルとのみをコピー	|
